# BNMV Website

This is a static website for the Better Nothing Music Visualizer app.

## Features
- Modern, Nothing-inspired design (Monochrome + Red accents).
- Fully responsive.
- Comparison table with stock features.
- Technical details (FFT, 12-bit depth).
- Device compatibility list.
- Direct links to GitHub Releases and Discord.

## How to Host
The easiest way to host this website is via **GitHub Pages**:

1. Go to your repository settings on GitHub.
2. Navigate to **Pages**.
3. Under **Build and deployment**, set the source to **Deploy from a branch**.
4. Select the branch and the `/website` folder.
5. Save, and your site will be live at `https://<your-username>.github.io/better-nothing-music-visualizer/`.

Alternatively, you can use services like Netlify, Vercel, or even just open `index.html` locally.
